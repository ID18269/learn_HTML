HTML

HTML (HyperText Markup Language) veb-sahifalar yaratish uchun ishlatiladigan markaziy tili hisoblanadi. HTML orqali matn, rasmlar, videolar, linklar va boshqa elementlarni sahifalarga qo'shish mumkin.

CSS

CSS (Cascading Style Sheets) veb-sahifalarni stilini belgilash uchun ishlatiladigan til hisoblanadi. CSS orqali sahifalarga ranglar, shriftlar, marginlar, paddinglar va boshqa ko'rinishlarni belgilash mumkin.

JavaScript

JavaScript veb-sahifalarga interaktivlik qo'shish uchun ishlatiladigan dasturlash tili hisoblanadi. Bu til orqali sahifalarda animatsiyalar, formalar, dinamik ma'lumotlar va boshqa interaktiv elementlar yaratish mumkin.

Responsive Design

Responsive design veb-sahifalarni qurilmalar va ekran o'lchamlariga moslashtirishni maqsad qiladi. Bu usul orqali sahifalar foydalanuvchilar tomonidan qo'llaniladigan har qanday qurilmada to'g'ri ko'rinadi.

Mobile App Development

Mobil ilovalarni yaratish uchun turli dasturlash tillari va platformalar mavjud. Bu orqali mobil ilovalar iOS va Android platformalarida ishga tushirilishi mumkin.
